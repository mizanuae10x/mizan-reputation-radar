# CODEX.md — Build Instructions for Codex CLI

## Project: رادار السمعة (Mizan Reputation Radar)
## Category: ai-tool

## Description
Real-time reputation monitoring engine. Tracks mentions across news, social media, and web. Sentiment analysis, alert system, and trend visualization dashboard.

## Features to Build
1. مراقبة 24/7
2. تحليل مشاعر متقدم
3. تنبيهات فورية
4. لوحة تحكم
5. تقارير PDF

## Tech Stack
Python, FastAPI, React, SerpAPI, Tavily, Claude API

## Design Guidelines
- RTL layout (Arabic first)
- Color scheme: Gold #D4AF37, Dark #1B1D21, Beige #f2eccf
- MOJ branding
- Mobile responsive
- Accessibility compliant

## Architecture
- Use Next.js App Router
- API routes in /app/api/
- Components in /components/
- Arabic + English i18n support
- Tailwind CSS for styling

## Priority Order
1. Core functionality first
2. UI/UX polish
3. API integration
4. Testing
