# رادار السمعة — Mizan Reputation Radar

> محرك مراقبة سمعة لحظي. يتتبع الإشارات عبر الأخبار والويب. تحليل مشاعر، نظام تنبيهات، ولوحة تصور الاتجاهات.

Real-time reputation monitoring engine. Tracks mentions across news, social media, and web. Sentiment analysis, alert system, and trend visualization dashboard.

## 🏛️ وزارة العدل — Ministry of Justice, UAE

## ✨ المميزات / Features

- مراقبة 24/7
- تحليل مشاعر متقدم
- تنبيهات فورية
- لوحة تحكم
- تقارير PDF

## 🛠️ التقنيات / Tech Stack

- FastAPI
- React

- Tavily
- Claude API
